Color Blender

Challenge: Color Blending

For this challenge, I wanted to experiment with color blending and discover all the different ways to blend color using p5.js. I played around with lerpColor() and hue to create different effects across my sketches. For this sketch, I literally created a color blender! I felt this was a good place to start using lerpColor() to find the halfway point between two colors. Adding in the blender graphic and juice effects just made the experience a little more fun.

Instructions:

  • Select two colors you want to blend together using the two color pickers. Press the reset button when done picking to update the blender colors.
  • Press and hold the button on the blender to mix the two colors. The button will turn green when fully mixed.
  • Press the reset button to reset the blender to the current color picker colors.
